# DeepSeek TUI 学习路径

从零到精通，掌握 DeepSeek TUI 的完整互动学习网站。

## 概览

- **5 个阶段**，**21 个实践练习**
- Next.js 16 + TypeScript + Tailwind CSS v4
- 全静态预渲染 (SSG)
- GitHub 风格深色主题 + 终端命令演示

## 关于

本项目由 **DeepSeek TUI**（v0.8.20）与用户对话交互生成，全程零人工编码。所有页面、组件、样式、部署配置均由大模型根据自然语言指令自动编写。

## 阶段

| 阶段 | 内容 |
|------|------|
| ⌨️ 基础交互 | 读写、搜索、Shell 命令 |
| 📋 结构化任务 | Plan + Checklist 多步骤管理 |
| ⚡ 并行与分治 | Sub-agent 并行执行 |
| 🎯 技能系统 | 自定义技能持久化 |
| 🏗️ 复杂工程 | 大规模重构、批量处理 |

## 本地运行

```bash
pnpm install
pnpm dev
```

打开 http://localhost:3000
